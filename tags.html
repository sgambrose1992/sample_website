Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

  1. Block Elements

    2. Tag Name Purpose
    • h1–h6
    		 headings include a heading (levels 1–6)
    • p
    		 paragraph include a paragraph of text
    • table
    		 table include a table
    • tr
    		 row include a row of data
    • th
    		 table header Make a table header
    • td
    		 table data include a table data cell

  2. Inline Elements

    3. Tag Name Purpose Example Result
    • em
    		 emphasized make emphasized text <em>technical sophistication</em> technical sophistication
    • strong
    		 strong make strong text <strong>at least a billion people</strong> at least a billion people
    • a
    		 anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
    • img
    		 image include and image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
    • code
    		 code display source code <code>rendered in a monospace font</code> rendered in a monospace font

  3. Document Tags

    4. Tag Name Purpose
    • html
    		 html defines the root of the document
    • head
    		 head defines information about the document
    • body
    		 body defines the document's body
    • title
    		 title defines the title of the document
    • meta
    		 meta define mata data about the document
    • !DOCTYPE
    		 !DOCTYPE defines the document type
    • br
    		 break enter line break
    • header
    		 header label the page header
    • blockquote
    		 block quotation show formatted quotation
    • !--...--
    		 comment defines a comment
